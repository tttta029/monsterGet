# モンスターコレクト - 2D Monster RPG

ブラウザで動作する本格2Dモンスター収集RPGです。

## 遊び方

1. `index.html` をローカルサーバー経由でブラウザで開く
2. 最初のパートナーモンスターを3体から選ぶ（ほのお/みず/くさ）
3. フィールドを探索してモンスターを捕まえ、育成しよう！

## 操作方法

| キー | アクション |
|------|-----------|
| W/↑ | 上移動 |
| A/← | 左移動 |
| S/↓ | 下移動 |
| D/→ | 右移動 |
| E | 調べる（NPCと会話） |
| M | メニューを開く |
| Enter/Space | 決定 / ダイアログ送り |
| Esc/X | キャンセル / メニュー閉じ |
| 1〜4 | バトル中：技選択 |
| R | バトル中：逃げる |

## ゲーム内容

### モンスター
- **70種類**（進化形含む）
- 18タイプ対応（ほのお/みず/くさ/でんき/こおり/かくとう/どく/じめん/ひこう/エスパー/むし/いわ/ゴースト/ドラゴン/あく/はがね/フェアリー/ノーマル）
- 単タイプ＋複合タイプ（2タイプ持ち）
- 6ステータス制（HP/攻撃/防御/特攻/特防/素早さ）
- **進化システム**（2段階進化あり）
- **161種類の技**（物理/特殊/変化）
- Lv5〜60で技習得（モンスター個別テーブル）

### マップ構成
- **町6つ**（各町にモンスターセンター完備）
- **フィールド15エリア**
- **モンスターセンター内部6つ**
- 合計27マップ

### マップ構造
```
はじまりの町 → みどりの草原 → いかずちの丘
                    ↓                ↓
              灼熱の谷 ← → 霧の森
                    ↓                ↓
            港町マリーナ        竜の山道
                    ↓
            烈風の峡谷 → 深淵の洞窟
              ↓                ↓
         禁断の火山      天空の都シエル → 風の回廊 → 雲海峡谷 → 雷鳴の遺跡 → 雷神の祠
                              ↓
                        嵐の荒野 → 暗黒の深森 → 深緑の隠れ里
                              ↓
                        灼熱の地底 → 終焉の頂 → 伝説の聖域
```

### バトルシステム
- ターン制コマンドバトル
- タイプ相性（18タイプ完全対応）
- 物理技/特殊技分岐ダメージ計算
- タイプ一致ボーナス（STAB 1.5倍）
- 複合タイプの倍率掛け算
- 0倍（無効）完全対応
- **トレーナーバトル**（17名配置、高額報酬）
- 逃走率80%（トレーナー戦は逃走不可）
- パーティ全員経験値共有

### 捕獲システム
- HPが低いほど成功率UP
- ボール3種（通常×1.0/スーパー×2.0/ハイパー×3.0）
- HP10%以下+ハイパーボール=80%保証
- パーティ満員時はボックスへ自動送り

### 進化システム
- レベル到達で自動進化判定
- 進化レベルを過ぎていても次回レベルアップ時に進化
- 2段階進化（ヒノコグマ→ブレイズベア→インフェルノ 等）
- 進化時にタイプ変化あり
- 進化演出付き

### 技習得システム
- Lv5刻みで固定テーブルから習得
- 4技満杯時はプレイヤーが忘れる技を選択
- 覚えない選択も可能
- 技管理画面で過去に覚えた技を再セット可能

### 施設
- **モンスターセンター**：回復/ボックス/ショップを統合
- **モンスターボックス**：上限なし、預け時全回復
- **ショップ**：数量選択UI（1/5/10/最大）、売却対応

### メニュー機能
- パーティ管理（並び替え/詳細ステータス/技管理）
- バッグ
- 図鑑（捕獲済み✓表示）
- タイプ相性表
- ワールドマップ（Canvas描画、BFS自動レイアウト）
- セーブ（localStorage）

### その他
- エンカウント：草むらのみ10%（道路/町は0%）
- バトル後7歩エンカウント無効
- フェードトランジション（マップ遷移/バトル出入り）
- スムーズカメラ＋プレイヤー補間移動
- プレイヤースプライト（方向別画像切替）
- 自動セーブ（バトル後）+ 手動セーブ（メニュー）
- セーブデータからの復元（タイトル画面「つづきから」）
- 全滅時は最後に回復した町へ自動復帰

## ローカルサーバーでの起動

```bash
# Python 3
python -m http.server 8080
# または
py -m http.server 8080

# Node.js
npx serve .
```

`http://localhost:8080` でプレイ。

## 技術構成

- HTML5 Canvas + Vanilla JavaScript
- データ駆動設計（JSON）
- 外部依存ゼロ
- ステートマシン方式バトルシステム
- BFS自動レイアウト（ワールドマップ）
- localStorage セーブ
