# モンスターコレクト - 2D Monster RPG

ブラウザで動作するシンプルな2Dモンスター収集RPGです。

## 遊び方

1. `index.html` をブラウザで開く（ローカルサーバー経由推奨）
2. 最初のパートナーモンスターを3体から選ぶ
3. フィールドを探索してモンスターを捕まえよう！

## 操作方法

| キー | アクション |
|------|-----------|
| W/↑ | 上移動 |
| A/← | 左移動 |
| S/↓ | 下移動 |
| D/→ | 右移動 |
| E | 調べる（NPCと会話） |
| M | メニューを開く |
| Enter/Space | ダイアログを閉じる |
| Esc | メニューを閉じる |

## ゲーム内容

- **モンスター**: 30種類（12属性）
- **マップ**: 町2つ + フィールド5つ
- **バトル**: ターン制コマンドバトル（相性システムあり）
- **捕獲**: HPが減るほど成功率UP
- **育成**: 経験値でレベルアップ

## マップ構成

```
はじまりの町 → みどりの草原 → いかずちの丘
                    ↓                ↓
              灼熱の谷    ←    霧の森
                    ↓                ↓
            港町マリーナ        竜の山道
```

## ローカルサーバーでの起動

JSONファイルの読み込みにはローカルサーバーが必要です。

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .

# VS Code: Live Server拡張を使用
```

## 技術構成

- HTML5 Canvas
- Vanilla JavaScript (フレームワーク不使用)
- データ駆動設計 (JSON)
- 外部依存ゼロ
